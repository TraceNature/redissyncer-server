# RedisSyncer

RedisSyncer是一个redis同步工具，应用于redis单实例及集群同步

## 项目目标
* 基于java编写服务化的redis同步服务；
* 基于redis迁移服务开发Cui及Gui可视化界面方便使用；
* 覆盖单实例多版本、多实例多版本、全量+增量、增量等多个场景；
* 满足客户迁移同步、缓存预热、redis灾备、集群扩缩容、redis版本升级等实际应用场景；
* 易于与现有云环境集成；
